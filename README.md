# 🧠 Pancreatic Tumor Detection using Deep Learning

## 📌 Overview

Pancreatic cancer is one of the most deadly cancers due to late detection. This project uses **Deep Learning techniques** to automatically detect pancreatic tumors from MRI images, helping in early diagnosis and improving survival rates.

---

## 🚀 Features

* 🔍 MRI Image Analysis
* 🤖 AI-based Tumor Detection
* 📊 Model Accuracy Evaluation
* ⚡ Fast and automated prediction system

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** TensorFlow / PyTorch, OpenCV, NumPy
* **Model Type:** Convolutional Neural Network (CNN)
* **Tools:** Jupyter Notebook / VS Code

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/ganesh-711/PancreaticTumorDetection.git
cd PancreaticTumorDetection
```

### 2️⃣ Create virtual environment

```
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```
